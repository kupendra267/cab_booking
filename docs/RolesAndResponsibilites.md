# Roles and Responsibilities – Cab Booking System

## User (Rider)
- Register and login.
- Book a cab.
- Track ride status.
- View ride history.
- Make payments.

## Driver
- Login to the system.
- Accept or reject ride requests.
- Update ride status.
- View earnings.
- View ride history.

## Admin
- Manage users.
- Manage drivers.
- Monitor bookings.
- Manage payments.
- View reports and analytics.
- Manage vehicle categories.

## Summary
This document describes the responsibilities of each user role in the Cab Booking System. Each role has specific permissions to ensure the application functions efficiently and securely.s
