# Features – Cab Booking System

## Main Features

### 1. User & Driver Authentication
- User Registration
- User Login
- Driver Login
- Secure Authentication

### 2. Cab Booking
- Enter Pickup Location
- Enter Drop Location
- Select Cab Type (Mini, Sedan, SUV)
- Fare Calculation
- Confirm Booking

### 3. Ride Management
- Assign Driver
- Live Ride Tracking
- Ride Status Updates
- Ride History

### 4. Payment
- Online Payment
- Cash Payment
- Download Receipt

### 5. Booking Management
- Schedule Booking
- Cancel Booking
- Reschedule Booking

### 6. Admin Features
- Manage Users
- Manage Drivers
- Manage Vehicles
- View Reports
- Monitor Bookings

## Summary
The Cab Booking System provides secure authentication, easy booking, live ride tracking, payment options, ride history, and an admin panel for complete management.
