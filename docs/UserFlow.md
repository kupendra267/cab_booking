# User Flow – Cab Booking System

## User Flow

1. User opens the Cab Booking application.
2. User registers or logs in.
3. User enters pickup and drop locations.
4. User selects a cab type.
5. System calculates the fare.
6. User confirms the booking.
7. System assigns the nearest available driver.
8. Driver accepts the ride request.
9. User tracks the driver's live location.
10. Ride starts.
11. Ride ends at the destination.
12. User completes the payment.
13. Ride details are saved in the ride history.
14. User can view previous bookings and receipts.

## Summary

The user flow describes the complete journey of a rider, from logging into the application to booking a cab, completing the ride, making payment, and viewing ride history.
